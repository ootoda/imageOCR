## はじめに

画像を読み込むだけで文字を自動抽出してくれるデスクトップアプリを作りました。

実行すると画像ファイルと同じ名前のテキストが自動的に出力されます。


## こんなことができます

・　ドラッグ＆ドロップ対応～画像ファイルをウィンドウにそのまま放り込むだけで読み込み開始。

・　画像のサムネイル表示～読み込んだ画像が左側に自動表示され、どの画像を解析中か一目でわかります。

・　高精度OCR（EasyOCR）搭載～日本語 / 英語を同時に認識。手書きや写真撮影でも比較的高い精度で抽出できます。

・　抽出テキストの自動保存～画像と同じフォルダに「画像名_txt.txt」として自動生成します。

・　フォントサイズ変更スライダー付き～抽出されたテキストを読みやすい大きさに自由に調整可能。

・　クリックでも選択可能～ファイルダイアログからの読み込みにも対応。

・　処理状況をステータスバーに表示～進捗や文字数がリアルタイムにわかります。



## 動作環境

* Python
* Windows / macOS / Linux　で動作します。
  （Tkinter GUI + easyocr + opencv が必要）



## インストール（必要なライブラリ）

前準備
1. Pythonのインストール
・　Python 3.8以上が必要です 
・　python.orgからダウンロード・インストール 
・　インストール時に「Add Python to PATH」にチェック 

各ライブラリの説明：
・　 pillow：画像処理 
・　 easyocr：OCR機能（日本語・英語対応） 
・　 tkinterdnd2：ドラッグ&ドロップ機能 
・　 opencv-python：画像読み込み処理 
・　 numpy：数値計算ライブラリ 


## 使い方

1. コマンドプロンプト（Windows）またはターミナル（Mac/Linux）で以下のコマンドを実行します:

```
pip install easyocr opencv-python pillow numpy tkinterdnd2
```

2. python imageOCR.py を実行（Windowsの場合はファイルをダブルクリック）初回起動は1～2分程度かかります（モデルのダウンロード） 
3. 画像をドラッグ＆ドロップ、または「ファイル選択」ボタンから読み込み
4. 数秒待つだけで右側にテキストが自動表示
5. 必要ならクリップボードコピー / 任意の保存も可能
6. 抽出内容は自動的にテキストファイルとして保存されます
